## 维护建议

- PDF 作为原始版本保留，不建议直接改名或移动。
- 新增需求、接口变更、字段变更优先同步到 Markdown 摘要，再回写正式设计文档。
- 如进入 V2，建议新增 `v2-scope.md`，单独沉淀用户分群、AB 实验、营销旅程、转化归因等能力。
